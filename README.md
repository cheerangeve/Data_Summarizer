# Data_Summarizer
Genesys Hackathon

# Dependencies
1) flask
2) Spacy
3) gensim
4) numpy
5) pandas
6) nltk

# How to run this extension
1) clone the repository
2) Load "ext" folder in chrome to unpack the extension
3) run the server.py to create the server
4) now use the extension in chrome to summarize the data
